*************** Installation *****************
----------------------------------------------
-edl firmware 10.0.18 (more recommended) - or 11.0.04 - 11.0.16
-persist oreo / or persist.img by fastboot mode (twice) -  (recommended/optional)
-boot to recovery
-format data->yes
-reboot to recovery
-wipe: system, data/cache, dalvik (optional)
-flash ROM (reboot system optional) - do this if using weebgapps
-flash custom recovery (you don't need a rom that comes with built-in recovery, example: msm-xr, havoc, syberia)
-reboot to recovery
-flash gapps  (optional) — Note: Use FlameGApps (more recommended); No use WeebGapps for Rom FBE
-flash kernel (optional)
-flash magisk  (optional)
-reboot to system
